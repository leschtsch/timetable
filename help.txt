Эта программа должна создавать расписание в автоматическом режиме.
	- Чтобы составить расписание, отредактируйте файл ввода "данные.xlsx", затем нажмите кнопку "составить расписание".
	- Чтобы открыть настройки, нажмите кнопку "открыть настройки".
Настройки содержат в себе настройки внешнего вида, а также настройки качества генерации и количества генерируемых расписаний.
Программа считывает и выводит данные через файлы типа .xlsx (файл Microsoft excel).
Файлы не должны быть открыты при работе.
Если, по каким-либо причинам, расписание создать не удастся, программа выведет соответствующее сообщение.
Причин, почему расписание нельзя составить может быть несколько:
	- в расписании не хватает места для пар
	- вы пытаетесь поставить две пары в одно место
	- вы пытаетесь поставить одного преподавателя в два места

Пример файла ввода есть в папке программы. Файл вывода появится после первого созданного расписания.
